This is my practice with git.

How to save changes to repository:

  1. To do the following, please update your code.
  2. Make sure you are in the current folder you are trying to update ex. supreme-train
  3. git status: shows all of the modified files
  4. git add: asks git to track added files. This can be followed by "." to track all new files/folders.
  5. git commit -m "what and why you are commiting" -m "description". Then we are ready to commit.
  6. git push: push to remote repo where project is hosted.
